== Description ==
Modern Bulk Editor for Media Library files, images, videos, Audio, etc., edit thousands of files in a spreadsheet inside wp-admin. You can import information from Excel, Export to edit in Excel or Google Sheets, etc.

= Use Cases =

* WooCommerce stores : you can view all the product images in a single place
* You want to edit product images quickly
* You want to search images by Keyword, alt text, caption, associated post or product
* You want to Edit Thousands of Images quickly using a spreadsheet
* You want to copy the associated post title to the alt text, caption, or viceversa

**Multiple editors**

You can edit in our spreadsheet inside wp-admin, so all the changes apply live.
You can also edit in Excel or Google Sheets and import the changes easily.

**Perfect for SEO edits**

You can view the image alt text, caption, title, file name, and descriptions in the spreadsheet, so you can edit the fields quickly.

You can copy the associated product title or post title to the Alt Text or captions, you can also copy the categories, tags, brands from the associated post title to the alt text, so you can create rich text for SEO purposes.

**Make Advanced Searches**

You can search by multiple fields and using multiple conditions.
For example, find all images without alt text or captions, etc.

You can make searches using any field and any search operator (<,>,=,<=, >=, LIKE, NOT IN, etc.).

**Edit thousands of images and files at once**
We have a formulas engine that lets you update a lot of files quickly.

You can replace values in ANY field:
- Replace brand names in descriptions
- replace words in captions or descriptions in all the products

You can combine fields:
- Copy the "associated post title" to the "alt text" field
- And more

== Installation ==
= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type the plugin name and click Search Plugins. Once you’ve found our plugin you can install it by simply clicking “Install Now”.

= Manual installation =
The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here.](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation)


== Frequently Asked Questions ==

= Can I use this plugin on cheap / shared servers? =

Yes. You can set up the number of posts to save per batch and the waiting between batches. Tweaking those settings you can make it work with any server.
